# 引言

**HTML由文本、图像、链接、多媒体等一系列标签元素组成，其中文本是HTML重要组成部份，而大部分结构化文本如报纸杂志新闻等都是由标题和段落组成的，所以我们先来学习标题和段落标签**

# 标题标签

当我们浏览网页的时候，标题是非常重要的，它们能够帮助我们快速地找到我们想要的内容

在 HTML 中，一共有六个等级的标题标签，从 h1 到 h6，每个等级的标题都有不同的重要性。例如，h1 标签用于表示最重要的标题，而 h6 标签用于表示最不重要的标

让我们来看一下如何使用标题标签。假设我们有一个网页，我们想要在网页中添加一个 h1 标题，可以使用下面的代码：

```html
<h1>这是一个h1标题</h1>
```

这里，我们使用了 h1 标签来定义一个标题，它的内容是 "这是一个h1标题"。需要注意的是，h1 标签是双标签，所以需要使用起始标签和结束标签来包含标签的内容。

除了 h1 标签，我们还可以使用 h2、h3、h4、h5 和 h6 标签来定义不同等级的标题。例如，下面的代码定义了一个 h2 标题：

```html
<h2>这是一个h2标题</h2>
```

需要注意的是，每个标题都应该独占一行，这样可以使页面更加清晰易读。

在代码中添加标题标签后，可以通过刷新页面来查看效果。这样我们就可以看到添加的标题以及它们在页面中的效果了。

h3 到 h6 标题依次表示更小的标题，用于表示页面内容的更具体的部分或子标题。它们的大小和重要性逐渐减小。通常，它们被用于表示页面中的列表、段落、注释、引用等等。

```html
<h3>这是一个h2标题</h3>
<h4>这是一个h2标题</h4>
<h5>这是一个h2标题</h5>
<h6>这是一个h2标题</h6>
```

总之，标题标签是非常有用的工具，可以帮助我们更好地组织页面内容，同时也能让页面更加易读和易用。

<aside>
💡

- 最好只对每个页面使用一次 <h1>——这是顶级标题，所有其他标题位于层次结构中的下方。
- 请确保在层次结构中以正确的顺序使用标题。不要使用 <h3> 来表示副标题，后面再跟 <h2> 来表示二级副标题——这是没有意义的，会导致奇怪的结果。
- 在现有的六个标题层次中，除非觉得有必要，否则应该争取每页使用不超过三个。有很多层次的文件（例如，深层次的标题层次）会变得笨重，难以浏览。在这种情况下，如果可能的话，建议将内容分散到多个页面
</aside>

# 段落标签

在 HTML 中，段落是由 **`<p>`** 元素标签定义的。

例如，下面的代码定义了一个段落：

```
<p>这是一个段落。</p>
```

浏览器会将该段落显示为一个独立的块元素，并在段落前后添加一个空行。

## **段落标签的语义意义**

段落标签不仅仅是为了实现样式上的排版，它还有语义上的意义。

段落标签告诉浏览器和搜索引擎，这部分文本是一个独立的段落，而不是一个连续的文本块。这使得搜索引擎能够更好地理解你的网页结构和内容，并对其进行更准确的索引。

此外，使用段落标签可以使你的文档结构更加清晰和易于阅读。如果你只是将文本写在一起，而没有使用段落标签，那么阅读起来就会非常困难。

## **段落标签的嵌套**

段落标签可以嵌套在其他元素中。例如，你可以在列表项中使用段落标签：

```
<ul>
  <li><p>第一项</p></li>
  <li><p>第二项</p></li>
  <li><p>第三项</p></li>
</ul>

```

## **总结**

当使用h1和p标签时，它向读者和搜索引擎表示该段文字是一个独立的标题或者段落。这种语义化的信息使得搜索引擎可以更好地了解网页的结构和内容，以及从段落中提取更准确的信息，从而提高搜索引擎优化的效果。同时，这种语义化的信息也使得使用屏幕阅读器等辅助技术的人士可以更方便地理解和导航网页内容。
